Consultation Only — reversible field-hiding, banner, and release exclusion

Repurposes the existing External consultation required field (no new SharePoint column) into a "this item will never release on a Thursday, it's a discussion placeholder" flag. Design mockup: Option 1 (dashed border + banner), confirmed.

The core design principle, and why it's fully reversible: every change below is a Visible formula reading the checkbox live — nothing is ever cleared, reset, or deleted when the box is ticked. Untick it and everything reappears exactly as it was (blank if it was never filled in, whatever was there before if it was). None of the Patch() save calls change at all — they already write whatever's currently in each control regardless of whether it's visible, so hidden fields keep their values in the background the whole time.

9. Bulk publish (PublishAllBtn) — skip consultation-only items in the count, the button text, and the actual patch

Three separate places on the same control all currently count/act on every unpublished item in the group — each needs the same extra condition so a Thursday group containing only consultation-only items doesn't show "Publish All (0)" for nothing, and so a mixed group's bulk action never accidentally publishes a discussion-only item.

Text, full replacement:

="Publish All (" & CountRows(Filter(ThisItem.GroupedItems, Published = false && 'External consultation required' = false)) & ")"

Visible, full replacement (only the count's Filter condition changed, staff-check and date-check lines are unchanged):

=!IsEmpty(Filter(colBossButtonUsers, Lower(Email) = Lower(User().Email))) &&
ThisItem.PlannedDate <= Today() &&
CountRows(Filter(ThisItem.GroupedItems, Published = false && 'External consultation required' = false)) > 0

OnSelect, full replacement (the ForAll now only patches non-consultation-only items, and the count notification matches):

=// PublishAllBtn.OnSelect
Set(varPublishGroupDate, ThisItem.PlannedDate);
Set(varPublishGroupCount, CountRows(Filter(ThisItem.GroupedItems, 'External consultation required' = false)));
ForAll(
    Filter(ThisItem.GroupedItems, 'External consultation required' = false),
    Patch(
        'Policy Proof Tracker',
        LookUp('Policy Proof Tracker', ID = ID),
        {
            Published: true,
            'Published Date': varPublishGroupDate
        }
    )
);
Refresh('Policy Proof Tracker');
Notify(
    "Published all " & varPublishGroupCount & " items for " & Text(varPublishGroupDate, "d mmm yyyy"),
    NotificationType.Success
)
